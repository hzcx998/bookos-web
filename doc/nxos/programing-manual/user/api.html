API设计

API应用程序接口

本文档存放的是用户程序的编程接口的设计说明。

NXOS的API接口分为几个大类。进程，线程，内存管理，时间管理，枢纽系统，文件系统，设备管理，网络管理，图形管理，安全管理，用户态驱动框架。

退出进程 void ProcessExit(int exitCode)
终止进程 int ProcessTerminate(int pid)
获取进程ID int ProcessGetId(void)
等待进程退出 int ProcessWait(int pid, int timeout, int flags)
